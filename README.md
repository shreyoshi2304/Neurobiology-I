# Neurobiology I: From Ion Channels to Computational Models

## Overview

This repository contains coursework and explorations from the Neurobiology I course, focusing on the fundamental principles of neuronal function.

The work covers both the biological basis of neural activity (such as voltage-gated ion channels) and its quantitative description through mathematical models.

## Motivation

A key highlight of this course was studying the Hodgkin–Huxley model, which provides a quantitative framework for understanding how neurons generate electrical signals.

Engaging with this model shaped my interest in computational neuroscience, particularly in understanding how complex neural behavior emerges from underlying biophysical mechanisms.

## Topics Covered

* Voltage-gated ion channels (Na⁺, K⁺ dynamics)
* Membrane potential and action potentials
* Conductance-based models of neurons
* Introduction to the Hodgkin–Huxley framework

## Future Directions

* Implement and simulate the Hodgkin–Huxley model in detail
* Explore neural population dynamics and network models
* Study stochasticity and variability in neural systems
* Connect single-neuron models to system-level behavior

## Note

This repository began as coursework but evolved into a deeper interest in computational neuroscience, particularly in modeling neural systems using quantitative and data-driven approaches.
